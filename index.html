AR6 Taiwan Climate 3D Maps

This page embeds three interactive PyDeck maps for Rx1day. Replace the iframe sources with another exported variable such as tx90p or prcptot after you run python src/make_3d_map.py with a different CSV file.

Future Minus Baseline
