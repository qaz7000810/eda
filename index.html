DATA SCIENCE MIDTERM PROJECT

AR6 臺灣氣候變遷 3D 地圖展示

本網站以 TCCIP AR6 氣候變遷關鍵指標資料為基礎，整合 Rx1day 與 TX90p 的空間視覺化與統計圖表，呈現觀測基期、未來高排放情境，以及 未來相對基期的變化。

研究說明 地圖展示 統計圖表 初步結論 後續方向

Overview

研究背景與資料來源

研究聚焦於極端降雨與高溫指標在臺灣的空間分布差異，並以 SSP5-8.5_2081-2100 作為未來情境示範。資料使用 TCCIP AR6 氣候變遷關鍵指標 CSV，將無效值 -99.9 轉換為 NaN 後進行分析。

研究問題

臺灣各地在未來高排放情境下，極端降雨是否有顯著增加？暖晝天數與 極端降雨之間是否存在可觀察的關聯？

核心指標

目前網站以年最大一日降雨量 Rx1day 與暖晝天數 TX90p 為主，後續可 延伸到 PRCPTOT、SDII、CDD、CWD 與 HWDI。

視覺化方法

地圖採用 PyDeck 的 2D 與 3D 圖層，統計圖使用 Plotly 輸出成獨立 HTML，方便直接嵌入 GitHub Pages。

Findings

初步發現

以下內容作為期中報告展示用的初步觀察，後續仍可配合更多指標與行政區尺度 聚合結果進一步深化。

空間差異明顯

從 2D 與 3D 地圖可快速辨識高值區與低值區，顯示極端降雨在不同地形與 區域條件下存在明顯差異。

情境差異可比較

SSP 比較圖與 boxplot 提供跨情境的整體分布視角，有助於理解未來不同排放 路徑下的平均值與變異程度。

跨指標關係值得追蹤

TX90p 與 Rx1day 的散佈圖可作為高溫與極端降雨關聯的起點，後續可加入更多 時段與情境檢視穩定性。

Next Steps

後續研究方向

下一階段可將更多指標與行政區層級分析納入展示，並補強模型不確定性與區域 差異的討論。

擴充指標

加入 PRCPTOT、SDII、CDD、CWD、HWDI 等指標，補足降雨與熱風險面向。

加強比較維度

比較不同時段與不同情境，並加入區域平均或行政區統計摘要。

深化報告敘事

結合文獻回顧、政策意涵與方法限制，讓網站不只展示圖，也能支撐論述。

網站使用 GitHub Pages 靜態部署，所有地圖與圖表皆以相對路徑載入 output/maps/ 與 output/charts/ 內的 HTML。
